🎨 ASCII Art Converter – Конвертер изображений в ASCII-арт
Превращайте любые изображения в удивительный ASCII-арт!
Поддерживает цветной вывод, разные наборы символов, настройку размера, инверсию и пакетную обработку.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Конвертация изображений – из PNG, JPG, BMP, TIFF, WEBP в ASCII-арт.

Настройка символов – выбор набора символов (по умолчанию, простой, детальный, блочный).

Цветной вывод – поддержка 24-битных цветов в терминале (ANSI-коды).

Масштабирование – автоматическая подгонка под ширину терминала или ручная настройка.

Инверсия – инвертирование яркости для тёмного/светлого фона.

Сохранение в файл – результат можно сохранить в текстовый файл.

Пакетная обработка – конвертация всех изображений в папке.

Прогресс-бар – наглядный индикатор выполнения.

Кроссплатформенность – работает на Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <вход> [опции]
Аргументы
Аргумент	Описание
<вход>	Входной файл или папка с изображениями
Опции
Опция	Описание
-o, --output <файл>	Сохранить результат в файл
-w, --width <N>	Ширина ASCII-арта (в символах)
-h, --height <N>	Высота ASCII-арта (в символах)
-c, --charset <набор>	Набор символов: default, simple, detailed, block
-i, --invert	Инвертировать яркость
-n, --no-color	Отключить цветной вывод
-b, --batch	Пакетная обработка (выходная папка)
-v, --verbose	Подробный вывод
-h, --help	Справка
Примеры
bash
# Конвертировать изображение в ASCII (авто-ширина)
python ascii_art.py photo.jpg

# Сохранить в файл с шириной 80 символов
python ascii_art.py photo.jpg -o output.txt -w 80

# Цветной ASCII-арт с детальным набором символов
python ascii_art.py image.png -c detailed -n

# Пакетная обработка всех изображений в папке
python ascii_art.py ./images -b ./ascii_output

🧠 Алгоритм работы
Загрузка – чтение изображения в память.

Масштабирование – изменение размера до целевой ширины/высоты (с сохранением пропорций).

Преобразование в оттенки серого – вычисление яркости каждого пикселя.

Маппинг – сопоставление яркости (0–255) с символами из выбранного набора.

Вывод – печать строк в терминале с цветами (если включено).
